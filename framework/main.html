neighborhood map

This was my sixth project in this class. I utilized the Google Maps API to display a map of the Denver area and then placed 10 markers of some of my favorite restaurants. I added the "bounce" animations to the markers when clicked as well as provided info windows for additional information about the restaurants. The default color of th markers were updated as well as a color change when hovered over. A sidebar was added to provide a list of the restaurants which would also generate the marker animations and info windows when clicked. The sidebar also included a dropdown menu that would filter the list of restaurants and their markers by category. The restraunt category for the filter was provided through the Foursquare API.
